# react-boilerplate (updated 2017.01.23)

You want a perfect react boilerplate right, maybe it is!

## React Ecosystem
+ react
+ redux
+ react-redux
+ redux-actions
+ react-router
+ react-router-redux
+ redux-saga
+ react-css-modules

## Engineering
+ webpack2
+ babel
+ react-hot-loader

### how to run

```
// start the express api server
npm run serve

// start the client use webpack dev server
npm start
```
